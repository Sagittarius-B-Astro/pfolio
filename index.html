Mars Airborne Residue Remediation System
In my ongoing senior Capstone Project, we test an electrostatic precipitator (EP) as a means of removing Martian dust from suits and equipment. I primarily investigated the electrical requirements of the EP.
AquaMetric
This project provides a means of getting accurate water output from a water filter tank by modeling fluid mechanics equations using sklearn, visualizing using matplotlib, designing a 3D-printed enclosure, and basic electronics.
Options Pricing and Greeks Calculator
Using streamlit, seaborn, matplotlib, this online app allows the user to calculate the ideal price for a call and put given various user inputs, as well as the calculation of the implied volatility, and every Greek I could find.
